첫만남이용권이 뭔가요?

첫만남이용권은 출생 아동의 초기 양육 부담을 덜어주기 위해 정부가 지급하는 바우처예요. 국민행복카드에 포인트로 지급되어 가맹점에서 카드처럼 결제할 수 있어요.

2024년 1월 1일 이후 출생 아동부터 첫째아 200만원, 둘째아 이상 출생아는 300만원이 지급돼요. 사용기간은 출생일로부터 2년간으로, 사용하지 않은 포인트는 자동 소멸되니 기간 내 꼭 사용하세요!

신청 방법
이렇게 신청하세요
출생일로부터 2년 이내 신청
  • 방문 신청: 아동 주민등록 주소지 읍·면·동 행정복지센터
  • 온라인 신청: 복지로(bokjiro.go.kr) 또는 정부24(gov.kr)
  • 국민행복카드가 이미 있으면 추가 발급 없이 바로 포인트 지급
  • 국민행복카드가 없으면 카드 발급 후 신청 (삼성·KB국민·신한·BC카드 등)
복지로 bokjiro.go.kr
정부24 gov.kr

잔액 조회 방법 (카드사별)

  • 삼성카드 앱 → 전체 → 정부지원사업 → 국가바우처 → 첫만남이용권
  • KB국민카드 앱 → 카드 → 국민행복카드 선택 → 첫만남바우처 한도 조회
  • 신한카드 홈페이지 → 혜택 → 정부지원사업 → 첫만남이용권
  • BC카드(페이북) 앱 → 전체 → MY바우처 → 첫만남이용권
  • 공통: 사회서비스 전자바우처 포털(socialservice.or.kr) — 단, 실시간 아닐 수 있음
💡 첫만남이용권 알뜰하게 쓰는 꿀팁
  • 산후조리원 비용이 크면 → 첫만남이용권으로 먼저 결제!
  • 대형 유아용품 매장(베이비빌리, 맘스맘 등) 방문 전 가맹점 여부 확인
  • 온라인 쇼핑몰(쿠팡, 네이버쇼핑 등)에서도 사용 가능한지 결제창 확인
  • 사용 만료일 1개월 전 알림 문자 발송 — 확인 후 잔액 사용
  • 카드사 제휴 할인과 중복 적용 가능한 경우 있음 — 카드사 앱 확인!
  • 결제 전 점원에게 "국민행복카드 바우처 되나요?" 꼭 확인

자주 묻는 질문 (FAQ)

Q. 첫만남이용권은 현금으로 바꿀 수 있나요?
안 돼요. 국민행복카드에 등록된 바우처 형태라 현금 인출이나 다른 카드로 이체는 원칙적으로 불가능해요. 반드시 가맹점에서 카드 결제 방식으로만 사용할 수 있어요.
Q. 사용 기간이 지나면 어떻게 되나요?
출생일로부터 2년이 지나면 남은 포인트는 자동 소멸돼요. 사용종료일 1개월 전에 문자 알림이 발송되니 꼭 확인하고 잔액을 소진하세요!
Q. 대형마트에서 식료품을 살 때도 사용 가능한가요?
가맹점으로 등록된 마트라면 가능해요. 단 해당 마트가 국민행복카드 가맹점인지 먼저 확인해야 해요. 일부 대형마트는 가맹점이 아니어서 결제가 거절될 수 있어요. 성인용품이나 주류만 구매하는 경우는 사용이 제한될 수 있어요.
Q. 온라인 쇼핑몰에서 사용하려면 어떻게 하나요?
결제 단계에서 국민행복카드를 선택한 뒤, 바우처 결제 옵션이 뜨는지 확인하세요. 해외 사업자 쇼핑몰이나 바우처 결제를 지원하지 않는 플랫폼은 사용이 불가해요. 대형 국내 쇼핑몰은 대부분 가능하지만 결제창에서 반드시 확인하세요.
